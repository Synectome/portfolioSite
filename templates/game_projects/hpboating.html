Hyde park B.I.A. boat racing

A 3D boat racing game with live leaderboard, created in the summer of 2021 for the Hyde Park Business Improvement Association's annual Pond Festival. The game is rendered in a 'low poly' style. The game was created using the Godot game engine, and blender 3D modeling software. This game was released under my newly created Tiny Rhino Games brand.
